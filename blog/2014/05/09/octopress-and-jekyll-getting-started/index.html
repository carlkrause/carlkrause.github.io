Octopress and Jekyll: Getting Started

May 9th, 2014

Blogging is meant to be simple. The blog is the epitome of democratic notions of the internet: easy, spur-of-the-moment, widely distributable, customizable to any whim. But often determining which platform to use, which theme to utilize, how to organize content, etc. provides a high-enough barrier to participation to most. Tens of millions of abandoned blogs are testament to this. Sure, means to express oneself are fragmenting further with each passing year, but blogs still function as a key off which creative-minded people present themselves to the world. Its utilitarian function is as important as ever.

Like many Millennials, the internet is strewn with my discarded blogs and social media profiles. Friendster still exists. So when I set up GitHub account I was heartened to learn that I could host a blog alongside my projects. Not another blog on another platform that I’d forget the password to then forget completely.

GitHub allows stingy users to host a site. But there’s also not a slick dashboard or much guidance for those who aren’t so computer literate. I didn’t want to reinvent the wheel, so I decided to use Jekyll and Octopress to get going.
